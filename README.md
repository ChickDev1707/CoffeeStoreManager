# CoffeeStoreManager
___

## GIỚI THIỆU

CoffeeStoreManager là phần mềm giúp chủ sở hữu hoặc người quản lý những quán cafe vừa và nhỏ quản lý quán một cách tiện lợi và dễ dàng hơn.


## ĐIỀU KIỆN
